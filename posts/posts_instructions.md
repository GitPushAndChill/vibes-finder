# Instructions to add posts

### Description
Adhere to the folowing data definition when populating the json file for adding new posts.

### Data elements
- **place**: the name of the place
- **city**: the city where the place is
- **vibe**: the vibes that the place is giving. only pick vibes that are available in the content/vibes.yml.
- **title**: a catchy title to give to the place optimized for SEO.
- **images**: cute pictures of the place and the vibe. Add minimal 1 picture and maximal 3 pictures in .jpg format.
- **short_description**: short description of the place and vibe, not longer than one sentence.
- **description**: description of the place and vibe, not longer than one paragraph.
- **coordinates**: coordinates of the place
- **adress**: the adress of the pace
- **created_on**: the timestamp when the post was created.
